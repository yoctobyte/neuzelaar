# License Note

Neuzelaar does not yet have a formal final license.

Current intent:

- this project is being developed and published in public
- the likely end state is a dual-licensing model in the spirit of `MIT OR GPL`
- that final decision has not been written down yet in a formal license text

Until that is settled, treat this repository as:

- shared for academic, research, and archival purposes
- visible development history by design
- not yet offered under a final production-ready legal license grant

Practical notes:

- Python and third-party library licensing constraints still need to be reviewed before a final top-level license is declared
- repository history, notes, tests, and design material are intentionally public as part of the project's development record
- public information on the internet tends to remain public; this note is not an attempt to pretend otherwise, only to avoid claiming a finished legal position before one exists

If you need a formal license for reuse, distribution, or commercial adoption, wait for the repository to gain an explicit final license file.
